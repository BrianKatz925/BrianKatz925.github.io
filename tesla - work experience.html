Tesla - Test Engineering

Summary

As a Test Engineer at Tesla, I am currently working on several components involved in the high voltage distribution systems within vehicles. In my role, I develop, plan, and execute several mechanical/electrical tests aimed at validating the reliability and design of components such as:

- Chargeports
- Latch Actuators
- High Voltage Connectors

Several tests involve the design of custom fixtures to validate component/system level strength, sealing, or environmental response. For extended duration tests, I implemented Python scripts to perform live test monitoring, data collection, automatic test control, and safety shutdown. To manage several tests at a time, I created test specification definitions using Polarion. This served as a common knowledge base for test and inspection procedures, which can be carried on into future programs.

Skills Used

- CAD for fixture design (CATIA V6, Solidworks)
- Scripting for test automation (Python, CAN)
- Test management, documentation, and spec definition (Polarion, Jira, Confluence)
